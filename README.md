# 📊 End-to-End E-Commerce Analytics Dashboard

## 🚀 Project Overview

This project is a multi-page interactive Power BI dashboard built using the Superstore Sales dataset. The dashboard focuses on sales, profit, product performance, regional analysis, and customer insights to support business decision-making.

---

## ✨ Features

* Executive KPI Dashboard
* Product Performance Analytics
* Dynamic Drill-Through Pages
* Custom Tooltip Pages
* Interactive Filtering & Slicers
* Advanced DAX Measures
* Business Insights & Storytelling

---

## 📌 Dashboard Pages

### 1️⃣ Executive Dashboard

* Total Sales
* Total Profit
* Profit Margin %
* YoY Growth
* Monthly Sales Trend
* Regional Analysis
* Top Products

### 2️⃣ Product Performance Analytics

* Highest Selling Product
* Profit by Sub-Category
* Sales vs Profit Scatter Analysis
* Top Product Sales
* Dynamic Insights

### 3️⃣ Product Detailed Analysis

* Segment Analysis
* Regional Performance
* Monthly Sales Trend
* Shipping Mode Analysis
* Profitability Breakdown

### 4️⃣ Custom Tooltip Page

* Interactive hover insights
* KPI mini dashboard
* Monthly trend popup

---

## 🛠 Tools & Technologies

* Power BI
* DAX
* Data Modeling
* Business Intelligence
* Data Visualization

---

## 📈 Key Insights

* Technology category generated highest revenue.
* November showed peak sales performance.
* West region contributed maximum profit.
* Some products showed high sales but lower profitability.
